Taurus UI 系列

Less fancy, more simplistic. Less flashy, more useful.

简介

Taurus UI 是由 Lipus Studio 制作的 PPT 仿操作系统（以下简称 PPT OS），隶属于【Lipus Studio PowerPoint UI Project】。

该 PPT OS 已收录至 PPT 世界【PPT OS 集锦】，遵循 CC 4.0 协议开源。

发展历程

  • 1.0

  • 2.x

  • Avalon

  • 更新日志

原名【Gemini OS】, 工作室最早于 2020 年春开始制作，2021 年 3 月初次发布 1.0 版预告，8 月份更名为【Gemini UI】。

1.0 正式版预告于 2022 年 1 月发布，2 月 2 日发布正式版演示视频，期间 UI 曾经历四次更迭。




2022 年 4 月末，工作室确定使用【Tuling 内核】开发 2.0 版本，通过引入该内核的 VBA 代码增强该作品内部功能的实用性以及减弱其臃肿性（1.0 本体就有 356 张幻灯片…）。

2022 年 6 月，2.0 Alpha 版的出厂引导界面先行预告发布，7 月发布正式版上线预告，预告发布的一段时间后，【Gemini UI】更名为【Taurus UI】，并于 8 月 20 日上线正式版。

2023 年 1 月，工作室将【Tuling 4】的部分新功能（诸如锁屏密码、本地文件管理器）引入 Taurus UI, 并于 1 月 29 日发布版本 2.5 Hydrogen。4 月 29 日发布的 2.6 版本正式将内核从 Tuling 3 升级至 Tuling 4。




Welcome to Avalon.

2023 年 3 月，工作室开始对 PPT OS 新作【Taurus UI Avalon】的制作，并于 3 月 4 日和 6 月 8 日分别发布出厂引导界面先行预告和初次进入桌面预告。该作品属于纯概念系统，计划不使用任何 VBA 代码（使动画可重复播放的代码除外）来模拟真实操作系统的部分功能。该项目目前处于停滞状态，计划最早于 2024 年 6 月末重启。（有可能会永久关闭，或者和此后计划开发的 Taurus UI 3.x 合并，因为工程量实在庞大…）




2.6.8 Hydrogen Official
1、修复了 UAC 窗口文字和按钮没对齐的 BUG。
2、修复了开发者选项切换效果不是平滑的 BUG。
3、修复了照明返回桌面平滑的问题。
4、修复了快应用 Desmos 图形计算器的一些 BUG。
5、修复了快应用启动动画的 BUG。
6、修复了快应用里的 Microsoft WebBrowser 下载补丁因为域名被污染而跳转到色情网站的问题。
7、减小了快应用的体积。
8、对部分地方字体进行更改。
9、弃用螃蟹云盘，换成奶牛快传。
10、对部分地方进行细微的调整。

2.6.5 Hydrogen Official
1、修复了时钟切换问题的 BUG。
2、修复了终端显示系统信息为 2.6 的 BUG。
3、修复了进入开发者选项时的调色板不统一的 BUG。
4、删除隐藏了某些无用的东西。
5、修复了使用自定义壁纸系统崩溃不会显示壁纸的问题。
6、修复了时间日期超格的问题。
7、对部分动画进行细微的更改。
8、修复了进入系统生硬的问题。
9、对锁屏输入密码进行了一些更改。
10、修复了文件信息描述位置不统一的问题。
11、更改了性能监视器的窗口颜色。

2.6.1 Hydrogen Official
1、修复了文档输入文档名出现错误的 BUG。
2、修复了手电筒颜色问题的 BUG。
3、修复了 BIOS 高亮颜色问题的 BUG。
4、对“设置-个性化”界面作出调整。
5、修复了“设置”部分界面的平滑问题。
6、删除了旧的 启动.bat

2.6 Hydrogen Offical
1、将 Tuling 内核升级为 Tuling 4。
2、修复了 WMP 打开文件报错的问题。
3、修复了从开始菜单打开应用的平滑问题。
4、对记事本进行修改。
5、修复了记事本大字报无法打开卡在UAC的问题。
6、对大字报设置进行调整，现在可以不用双击文本即可应用。
7、修复了 TRUMP 无法正常打开的问题。
8、修复了深浅色主题无法切换的问题。
9、整改时钟计时器（感谢 Fanbal 的帮助与支持）。
10、对文件-本地盘进行一些小修改。
11、修复许多超链接错误问题。
12、对系统壁纸进行修改。
13、支持修改锁屏文字字体颜色。
14、修复屏幕截图无法截屏的问题。
15、对 TRUMP 进行更新。
16、修复了开发者选项标题文字变白的问题。
17、修复了性能监视器的动画问题。
18、修复了 BIOS 设置按钮颜色的问题。
19、修复了 BIOS 字体颜色显示问题。
20、移除了旧版开始菜单。

2.5.2 Hydrogen Offical
1、删除 WMP 右上角的关闭按钮。
2、修复了一些交互问题。
3、修复了文件出现 Virtual Disk 的问题。
4、将补丁改成注册表形式。

2.5.1 Hydrogen Offical
1、对一些应用统一字体。
2、删去了未使用的因素。
3、修复了桌面自动跳转到设置的问题。
4、由于某些因素，取消掉了自动锁屏时长。
5、由于谷歌翻译在中国不在可用，在快应用里下架了谷歌翻译。
6、修复了文件应用磁盘显示错误的bug。
7、修复了螃蟹云盘无法正常打开的问题。
8、修复了一些拼写错误。
9、重写启动程序。

2.5 Hydrogen Official
1、引入了来自 Tuling 4 的锁屏密码和本地文件管理器功能。
2、新增了开机动画。
3、对锁屏 UI 进行调整。
4、由于生态圈关闭的问题，应用商店停止服务，更新通道更换为在“PPT 世界”里为 Taurus UI 设置的专用通道。
5、将 BIOS 添加至了使用向导里。
6、修复了系统分辨率获取的问题。
7、修复了在恢复模式下无法创建新备份的问题。
8、修复了无法打开屏幕截图文件夹的问题。
9、该版本起，字体不再嵌入系统（表现为初次进入系统，显示字体并不是系统所使用的字体）。所以用户需要安装字体才能获得完整体验。
10、将修复模式的 PowerV 版本更新至 4.10.1 Tolink。
11、更改了鸣谢名单。
12、修复了系统版本号错乱的问题。
13、将默认壁纸改回 Beach。

2.1.1 Hydrogen Official
此版本开始，Taurus UI 后的每一个大版本更新都会加上特定代号，2.X 代号定为“Hydrogen”。

2.1 Official
1、由于开发问题，“时钟” App 进入关于页和退出 App 的交互动画被计划删除。
2、对 BIOS 做了改动，将“关机”入口改为“电源操作”入口。
3、更改默认壁纸为 Morning (from Genshin Impact)。

2.0 Official
1、”Taurus UI” 名字正式启用。
2、对部分应用程序的交互和 UI 做了修改。
3、更换了终端图标。
4、对快应用模块的 UI 和图标做了修改。
5、给应用商店添加了生态圈。
6、修改了部分系统壁纸。
7、添加了鸣谢名单。

2.0 Beta
1、”开始菜单”改成窗口化。旧版开始菜单依然保留（有的人死了，他还活着）
2、更新了内部应用程序 UI。（有”关于”界面的应用的”关于”界面和进入动画被更改、设置和照片的首页 UI 被更改）
3、Tuling 3 的交互功能被重新启用，并在源系统基础上做了动画修改。
4、重新设计了”文件管理器” UI ，在内部添加了分类模块。
5、更新了系统内部应用程序的中等图标，图标颜色还可根据交互变化（这些图标均为自己设计及整合，并放在了系统文件内）。
6、”时钟”应用回归，计时器加载项更改。

2.0 Alpha
1、正式使用 Tuling 3.X 作为内核。
2、更新了系统和内部应用程序 UI。（任务栏调整、取消开始菜单和应用程序的模糊蒙版）
3、引入了诸多来自 Tuling 3.X 的特色功能（如：快应用、自定义壁纸、自定义用户名、屏幕截图、设置自动锁屏时长、BIOS 等等）以及应用程序（如：画板、计算器、终端等等）
4、控制中心全屏化，并添加了一些可操作的功能。

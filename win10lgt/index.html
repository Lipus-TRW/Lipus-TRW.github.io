如何给 Windows 10 披上 Longhorn 的皮套？

前言

我个人认为，如果当年 Longhorn 没有重置，而是沿着原先的轨迹继续开发的话，那么最终的结果一定比 Vista 还要华丽（但随之而来的也可能是比 Vista 更恐怖的谩骂 hhh）。
历史没有如果，但在科技领域里，凝固在过去的美丽，也可以重现于当下。虽说不够完美，但也足够惊艳。
这份主题包的作者来自 DeviantArt，可适用于 Win10 1903-22H2，是我个人见过的最好的主题包了。这里就教各位如何把这个主题包安装到系统上，重现 Longhorn 本来的样貌。

WARNING

注：访问 D 站需要魔法。

我已将美化主题所需的主题包和必备工具整理到一个压缩包上，大家可以根据包里给到的说明书来安装主题包噢~

必备工具

这里需要用到的工具有如下：SecureUxTheme、OldNewExplorer、OpenGlass、Aero Window Manager、Winaero Tweaker、StartIsBack++ 和 Rainmeter（皆已整理到我的压缩包里）。
（原主题包里也提到了 UltraUXThemePatcher 和 SecureUxTheme 二选一，但经过我个人实操，发现安装前者后主题包无法显效，而后者正常。所以个人建议使用 SecureUxTheme。）

安装步骤

1、安装 SecureUxTheme，之后重启电脑。

2、安装 OldNewExplorer（打开程序 OldNewExplorerCfg，之后点击窗口右上角的 Install 按钮，通过用户账户控制，完成安装），并按照如下图所示配置文件管理器顶栏样式，最后点击 Close 按钮关闭程序。之后打开文件管理器，查看更改效果。


3、打开我整理的主题包中的 “主题” 文件夹，将里面的所有文件直接复制粘贴到 “C:\Windows\Resources\Themes” 文件夹。之后再回到整个主题包的根文件夹中，找到 LHPRes.dll 文件，将其复制粘贴到 “C:\Windows\System32” 文件夹中 （注意全程要通过管理员权限）。


4、安装 OpenGlass 插件（将主题包中的 OpenGlass 文件夹直接复制到 C 盘根目录中，打开该文件夹，点击 install.cmd 文件安装插件，之后点击 startup.cmd 文件启用插件，等待插件下载静默好符号文件（不要关掉窗口）。下载完成后，若所有系统窗口标题栏已显现模糊效果，说明安装成功）。


5、安装 Aero Window Manager 插件并激活透明玻璃样式（过程较为复杂，还请耐心浏览）
（1）将主题包中的 AWM 文件夹直接复制到 C 盘根目录中。
（2）以管理员权限运行 CMD，执行如下命令。

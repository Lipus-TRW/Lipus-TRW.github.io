如何给 Windows 10 披上 Longhorn 的皮套？

前言

我个人认为，如果当年 Longhorn 没有重置，而是沿着原先的轨迹继续开发的话，那么最终的结果一定比 Vista 还要华丽（但随之而来的也可能是比 Vista 更恐怖的谩骂 hhh）。
历史没有如果，但在科技领域里，凝固在过去的美丽，也可以重现于当下。虽说不够完美，但也足够惊艳。
这份主题包的作者来自 DeviantArt，可适用于 Win10 1903-22H2，是我个人见过的最好的主题包了。这里就教各位如何把这个主题包安装到系统上，重现 Longhorn 本来的样貌。

NOTICE

注：访问 D 站需要魔法。

我已将美化主题所需的主题包和必备工具整理到一个压缩包上，大家可以根据包里给到的说明书来安装主题包噢~

必备工具

这里需要用到的工具有如下：SecureUxTheme、OldNewExplorer、OpenGlass、Aero Window Manager、Winaero Tweaker、StartIsBack++ 和 Rainmeter（皆已整理到我的压缩包里）。

NOTICE

（原主题包的说明书里也提到了 UltraUXThemePatcher 和 SecureUxTheme 二选一，但经过我个人实操，发现安装前者后主题包无法显效，而后者正常。所以个人建议使用 SecureUxTheme。）

安装步骤

WARNING

安装主题有风险，强烈建议对电脑数据做好备份，或者创建还原点以方便进行系统还原工作。

1、安装 SecureUxTheme，之后重启电脑。

2、安装 OldNewExplorer（打开程序 OldNewExplorerCfg，之后点击窗口右上角的 “Install” 按钮，通过用户账户控制 （注意有两次），并按照如下图所示配置文件管理器顶栏样式。最后关闭程序，打开文件管理器，查看更改效果，若效果如图所示，则证明修改完成。


3、打开我整理的主题包中的 “主题” 文件夹，将里面的所有文件直接复制粘贴到 “C:\Windows\Resources\Themes” 文件夹。之后再回到整个主题包的根文件夹中，找到 “LHPRes.dll” 文件，将其复制粘贴到 “C:\Windows\System32” 文件夹中 （注意全程要通过管理员权限）。


4、安装 OpenGlass 插件（将主题包中的 OpenGlass 文件夹直接复制到 C 盘根目录中，打开该文件夹，点击 “install.cmd” 文件安装插件，之后点击 “startup.cmd” 文件启用插件，等待插件下载静默好符号文件（不要关掉窗口）。下载完成后，若所有系统窗口标题栏已显现模糊效果，说明安装成功）。


5、安装 Aero Window Manager 插件并激活透明玻璃样式（过程较为复杂，还请耐心浏览）

（6）以管理员权限打开 “任务计划程序”（找不到的可以 Win+S 搜索该程序），在窗口右侧找到 “创建任务” 选项并点开，之后按照图示操作，将 AWM 设置为登录即运行的程序（这样可以保证主题永久生效）。








（7）将主题包中的 “Extras” 文件夹打开，再打开 “AWM Presets” 文件夹，安装其中任意一项注册表（二选一）。

之后以管理员权限运行 PowerShell，输入如下指令以重启 AWM，打开任意程序窗口查看渲染效果。若出现图示效果，则证明渲染成功。

（8）安装 Winaero Tweaker，之后在程序的搜索框中搜索 “title”，选择 “Advanced Appearance Settings/Window Title Bars”，按照截图所示将标题栏的高度调成 19，之后点击 Apply Changes 按钮应用效果，之后会弹出通知要求你注销电脑账户，重新登录（Sign out now），点击该按钮重新登录账户即可。

重新登录后，打开任意程序窗口。若看到标题栏高度如图示缩小，则证明渲染成功。

（9）安装 StartIsBack++，在弹出的配置窗口中，跳转到 “外观配置” 选项卡，调整任务栏图标大小为 “小”，最后点击 “应用” 按钮，应用任务栏效果。

（10）右键桌面，在弹出的菜单中选择 “个性化” ，进入设置界面。待界面加载好后，点击 “主题” 选项卡，找到我们先前安装好的主题包（默认选择 “Longhorn Pearl Normal” 主题），点击应用。稍等片刻后，若出现图示效果，则证明应用成功。


（11）进入 StartIsBack++ 的配置界面（打开被其修改过的开始菜单，鼠标右键菜单，选择 “属性” ，进入界面。若开始菜单还是 Win10 磁贴样式的，则右键开始菜单，找到 “属性” 选项并点击，进入界面。），选中名为 “LHPearl Normal + Windows 10 按钮” 的视觉样式，之后点击图中用红框圈出的 “添加” 符号，在弹出的窗口中定位到主题包的 “Extras\Start Buttons” 文件夹，选择任意一项开始菜单样式，之后点击新添加的样式，单击 “应用”。若出现图示效果，则证明应用成功。



（12）点击 SIB 配置界面中的 “开始菜单” 选项，根据自己的喜好对开始菜单作调整。

（13）进入主题包中的 “侧边栏” 文件夹，安装 Rainmeter 程序，之后点击文件夹中的 “LonghornSidebar.rmskin” 主题文件，并将其安装。这时会弹出高仿 Longhorn 原本样式的侧边栏，还有侧边栏的引导程序。之后根据自己电脑的实际情况选择适合的 DPI，点击 “CLOSE” 按钮跳到下个界面，最后单击 “OKAY I GET IT” 按钮，完成侧边栏的安装。



（14）若要使得窗口在全屏状态下，也能同时显示侧边栏，则可以使用主题包中的 “Aerial” 文件夹，打开其中的 “Aerial.exe” 程序，按照下图给出的样式配置程序，最后分别点击 “Apply” 和 “Hide” 按钮，应用效果。

至此，Win10 的 Longhorn 基本美化工作全部完成，尽情享用吧！

结语

除了安装主题的必备工具和主题包外，我整理的主题包中还包含了额外的内容（包括静态壁纸、极光壁纸、徽标、声效和鼠标指针样式。这些额外资源来自 Longhornize Windows 10 主题包，有兴趣的同学可以了解一下，不过需要魔法），可用作系统的进一步美化。有需要的同学可以食用。

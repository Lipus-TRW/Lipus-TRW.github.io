Windows Longhorn 10 Concept UI 系列

Return.Remake.

简单介绍

Windows Longhorn 10 Concept UI 系列由 Lipus Studio 制作，隶属于【Lipus Studio PowerPoint UI Project】。

该系列的作品有 Windows Longhorn 10 和 Windows Longhorn 10 Fluent。正如其名，Windows Longhorn 10 是将 Windows Longhorn 和 Windows 10 两个操作系统的部分界面特性结合起来的概念 UI。

Windows Longhorn 10 Concept Fluent

Windows Longhorn 10 Fluent 是 Longhorn 10 Concept 系列的一个分支作品，也是该系列的终止之作。该 PPT UI 在以往作品的基础上，在界面上加入了 Win10 Fluent Design 设计体系中的 Acrylic 元素，在颜色模式上加入了暗色模式。

如何给 Windows 10 套上 Longhorn Plex 的皮套？

前言

制作 Longhorn Pearl 主题包的作者 vaporvance，其实还仿造过 Aero 推出之前微软在 Longhorn 上所运用的 UI 效果——即 Plex，制作了另一款仿制 Longhorn 界面的主题。


而本人也有幸在某位朋友的帮助下，拿到了这款主题包。那么接下来，就让本人向各位详细介绍这个主题的安装教程吧~

NOTICE

1、该主题仅适用于英文镜像的 Windows 10 1903-22H2。（经检验，中文镜像后面改成英文语言的系统无法正常使用主题）
2、主题安装是一项复杂工程，没有足够耐心的话，我个人不建议折腾自己的电脑系统。
3、强烈建议在通读完教程之后，再决定自己是否要安装主题。
4、如果想要实装到实体机上，强烈建议先在虚拟机平台进行测试（我个人认为对系统镜像的要求就已经劝退一波人了 hhhh）。
5、安装主题前，创建还原点！创建还原点！创建还原点！

必装工具

1、UltraUXThemePatcher 或 SecureUxTheme
2、Aero Window Manager
3、VC_Redist 运行库
4、Windows Desktop Runtime
5、MacType
6、OldNewExplorer
7、Windhawk
8、StartIsBack++
9、Open-Shell
（上面列出的工具，除了 Windhawk 之外，我整理的主题包里都有附带）

基本美化

1、进入主题包内的 Programs 文件夹，安装好上面提到的必装工具（1、3、4、5、7、8、9）。


2、进入主题包内的 File Addons/Windows 文件夹，将里面的文件复制粘贴到 C 盘的 Windows 目录中。File Addons 文件夹里面，Windows 根目录里除 System32 文件夹之外的文件，迁移到 C 盘的 Windows 根目录中。System32 文件夹里的文件，就迁移到 C 盘 Windows 文件夹内的 System32 目录中。


3、进入主题包内的 Fonts 文件夹，安装里面附带的字体。

4、进入主题包内的 Libraries 文件夹，将里面的文件迁移到 C:\Users\用户名\AppData\Roaming\Microsoft\Windows\Libraries 目录中。

5、进入主题包内的 Programs 文件夹，将 OldNewExplorer 的程序包（文件夹）复制粘贴到 C 盘根目录。安装好程序后，按照下图所示勾选选项。


6、进入主题包内的 Programs 文件夹，将 AWM 和 PsTools 这两个文件夹迁移到 C 盘根目录中。之后，安装 Aero Window Manager（详细的安装过程，在上一期 Longhorn 主题安装教程的第 5.1-5.5 步）。

（注：如果标题栏长时间没有渲染，但是 CMD 如图所示，可以先进行下一步）
7、以管理员权限打开任务计划程序（Task Scheduler），找到程序界面右边的 “Import Task”，在弹出的文件选择窗口中，进入到主题包内的 Task Scheduler tasks 文件夹，导入附带的两项文件。


8、之后注销电脑帐户，重新登录，以确保渲染效果能够开机自动显示。

9、打开 Windhawk，搜索 Eradicate Immersive Menus 和 Resource Redirect 这两个模组并安装。安装完后，点击程序窗口右下角的 “Create a New Mod”，进入到代码编辑页面。之后，我们进入主题包内的 Windhawk mods 文件夹，使用记事本打开 “local@hide-search-bar-fork.wh.cpp” 这一文件，将里面的代码全部复制粘贴到 Windhawk 的代码编辑页面中。最后点击 “Compile Mod” 按钮，完成 Mod 的编译工作。



10、接下来使用 NSudo 来写入主题包内附带的必装的注册表文件。进入主题包内的 Programs/NSudo/NSudo Launcher 文件夹，以管理员权限运行 “NSudo.bat”，打开程序。我们找到 “Mode Settings” 这块区域，选择 Users 为 TrustedInstaller，并勾选 “Enable All Privileges” 选项，以获取最高权限。之后点击 “Browse” 按钮，找到注册表文件所处位置（主题包\registry\里面的子文件夹），选中注册表文件。随后点击 “Run” 按钮，无视弹出的警告窗口，写入注册表文件。
（注：registry 文件夹里面的所有注册表文件都要安装。）


安装完后重启系统，之后呈现如下效果。

11、进入主题包内的 Themes 文件夹，将主题文件复制到 C 盘的 Windows 主题文件夹（C:\Windows\Resources\Themes）中。之后直接点击 plex7.theme 文件，应用主题。


12、打开 StartIsBack 的配置程序，左侧栏点击 “Appearance”，选择 Win10 默认的开始菜单和 Plex 主题的任务栏，之后进入到 “Customize appearance” 区域，勾选 “Use a custom taskbar coloring”，之后点击选项右边的水滴图标，选择 “Blur”，再来到水滴右边的滑动条，把滑块拉到最左边。然后，鼠标移至窗口下方，找到 “Customize Taskbar Perks…” 并点击，在弹出的窗口中勾选 “Centered task icons”，点击 “OK”。


13、程序左侧栏切换到 “Switching” 选项卡，找到 “Where to show Start button” 区域，取消勾选区域内的这三个选择框，最后点击 “OK”，关闭程序并应用更改效果。之后，打开系统设置，进入 “个性化-任务栏” 设置，打开 “使用小任务栏（Use small taskbar buttons）” 按钮，再来到标题为 “合并任务栏按钮（Combined taskbar buttons）” 的下拉框，点击下拉框，选择 “从不（Never）”，以便在任务栏始终显示应用程序名称。


14、进入主题包内的 Open-Shell Config 文件夹，将名为 “retro” 的文件夹复制粘贴到 C 盘根目录中。再回到 Open-Shell Config 文件夹，将 “Plex.skin” 文件复制粘贴到 Open-Shell 的程序文件包中的皮肤文件夹中（默认目录是 “C:/Program Files/Open-Shell/Skins”）。


15、从开始菜单的 Open-Shell 文件夹中，打开 Open-Shell Menu Settings 程序，在弹出窗口的下方找到 “Backup”，点击它并选择 “Load from XML File…”。之后，在弹出的文件选择窗口中，选择 “主题包/Open-Shell Config/Menu Settings.xml” 文件，点击 “Open”，为开始菜单应用主题。



16、从开始菜单的 Open-Shell 文件夹中，打开 Classic Explorer Settings 程序，用和上一步同样的操作，应用 “主题包/Open-Shell Config/Explorer Settings.xml” 文件，以修改文件管理器部分 UI 效果。


17、进入主题包内的 Favourites Bar 文件夹，将里面的 exe 程序复制粘贴到 C 盘的 Windows 文件夹中，之后重启电脑系统。重启后若发现文件管理器顶栏出现如图所示的收藏栏，则说明操作成功。



至此，Plex 主题美化的基本工作结束，接下来的步骤是对系统的进一步美化。

进阶美化

图标更换

NOTICE

这一步建议在 Windows PE 环境下完成。

Plex 主题的图标文件放置在主题包的 Icons 文件夹中，将这一文件替换掉 Windows 默认的图标文件便可完成图标更换（位置在 C 盘 Windows 文件夹的 System32 目录中）。





游戏管理器安装

这里我们使用 Windows 7 Games for Win8/8.1/10/11 程序和由大神 aubymori 制作的 GameUX 程序来完成这项步骤（后者会放在主题包的 Programs 文件夹内）。


1、安装 Win7 Games。

2、将 GameUX 的程序包迁移到 C 盘根目录中，之后以管理员权限运行 “INSTALL.cmd” 文件，根据指示安装程序。安装完后，命令行会给你提供一串指令。我们将这串指令复制粘贴到文件管理器的地址栏中，键盘按下 Enter 键后，游戏管理器便就此弹出。为方便访问，我们可以将游戏管理器固定到文件管理器顶部的收藏栏中（点击顶栏右上角的 “Favourites” 按钮，选择 “Add to Favourites”。）

3、但是，或许是因为优化问题，管理器的 UI 布局比较杂乱。要解决这个问题，我们需要打开 Windhawk 里的 Resource Redirect 模组，点击 “Settings” 标签页，下拉到名为 “Theme path” 的区域，在这里，我们将主题包里 Windhawk mods 文件夹中的一个名为 “Plex_RR” 的文件夹的完整路径，复制粘贴到 Theme path 标题下方的地址栏中。之后我们再下拉到名为 “Redirect all loaded resources” 的区域，把这里的开关打开，然后点击 “Save Settings”，保存设置。最后打开游戏管理器查看效果。




侧边栏安装

我们需要先下载好 Rainmeter 这一程序来激活侧边栏插件。安装好程序后，进入主题包内的 Programs 文件夹，点击 “Plex_.rmskin” 文件。配置窗口弹出后直接点击 “OK” 即可。此后侧边栏就会加载。


开始菜单自定义说明

Open-Shell 的开始菜单自定义程序（Open-Shell Menu Settings > Customize Start Menu）个人认为实在太不友好了，因为稍有不慎你的菜单就会变得乱七八糟。下面我将根据自己的个人经验，来教各位如何把自己的开始菜单调教好。
1、打开开始菜单内程序菜单，右键任意程序，选择 “Pin to Start menu(Open-Shell)”，便可将该程序固定到开始菜单。

2、在自定义程序中，右键或者拖动右边的组件到左边，便可将其固定到开始菜单。若要改变该组件在开始菜单的位置，可直接将其上下拖动。

3、右键左边的组件，可以进行编辑组件属性、为其创建子菜单、删除该组件等操作。

4、使用 SEPARATOR 组件，可以在两个相邻对象之间添加分隔线。

5、将 COLUMN BREAK 组件放在两个相邻对象之间，可将开始菜单从单列变成双列，组件下方的对象则被放置在另一列的最前面。


6、此外，在 Programs 组件下方添加组件，开始菜单也会变成双列，该组件也会被放置在另一列的最前面。

7、将 COLUMN PADDING 组件放在某个对象上方，该对象就会被放置在开始菜单底部。

8、若想恢复开始菜单原本的样式，我们可以重复基本美化操作中的第 15 步。

9、若想访问 Win10 默认的开始菜单，我们可以在配置程序中搜索 “Start”，勾选 “Show Start screen shortcut”，点击 “OK”。之后打开开始菜单，便可以看到访问 Win10 开始菜单的快捷方式，点击它就能访问 Win10 默认的开始菜单了。



10、此外注意一点，在应用好作者提供的配置文件后（也就是基本美化操作中的第 15 步），不建议把上面的 “Shutdown dialog” 组件删去。因为这一组件是经过作者调教好的，没有图标也没有文字，可以完美隐藏在显示着关机和注销按钮的贴图里面。如果删去，换上右边组件库中的 “Shutdown dialog”，这一片区域就会多一层关机图标，而且纵使通过编辑组件隐藏了图标、删掉了标签，这片区域上还是会显示 “” 的字样，非常影响观感。



那么，Longhorn Plex 的美化教程就到这里。欢迎所有有问题的同学在评论区留言，或者去 B 站私信我噢。若我的讲解有任何问题，欢迎批评指正！

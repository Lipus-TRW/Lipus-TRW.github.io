Hill OS 现已归来！

是的，你没有听错，Hill OS 回来了！

开发历程

Hill OS 是由 JesseGary 凌禄 制作的 PPT 仿操作系统，首次发布于 2019 年 7 月。在第一个版本发布的 6 个月后，雨辰_Dream 接管了该产品的开发权，但长期未发布更新。
2022 年 3 月，蝶影（现为 IDTX Net.）获得了 雨辰_Dream 的授权，制作了 Hill OS X。后来该版本从 Hill OS 中独立出来，成为 IDTX Net. 的旗下品牌。 而 Hill OS 本体在此后的 3 年内未发布过任何更新，项目长期处于停滞状态（Hill OS 13 除外）。
2025 年 7 月，神秘人X 接管了 雨辰_Dream 的开发权限，开始为 Hill OS 制作更新。

· Hill OS X 见面了！（Hill OS X 宣传视频）

新版特点

新版 Hill OS 在原有版本（Hill OS 6.1）的基础上做出了大量改进。除了删除旧版本中的部分不合理设计，和添加新功能之外，该版本对许多地方的设计作出了美化和统一（例如字体方面）。
在旧版本中出现的诸多问题，也大多在新版 Hill OS 中得以解决（比如部分图标文字位置没对齐/错乱，动画速度太慢，平滑混乱等问题）。











WARNING

·部分旧版 Hill OS 的功能在新版中已被删除（例如设置里的 Chrome 密码管理，安卓设置应用和 Windows 子系统）。
·目前新版仍处于开发阶段，部分功能并不完善，使用过程中可能遇到少量体验问题，敬请各位用户留意。

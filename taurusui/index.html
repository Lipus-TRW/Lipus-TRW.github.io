Taurus UI 系列

Less fancy, more simplistic. Less flashy, more useful.

简介

Taurus UI 是由 Lipus Studio 制作的 PPT 仿操作系统（以下简称 PPT OS），隶属于【Lipus Studio PowerPoint UI Project】。

该 PPT OS 已收录至 PPT 世界【PPT OS 集锦】，遵循CC BY-SA 4.0 协议开源。

发展历程

  • 1.0

  • 2.x

  • Avalon

  • 3.x

原名【Gemini OS】, 工作室最早于 2020 年春开始制作，2021 年 3 月初次发布 1.0 版预告，8 月份更名为【Gemini UI】。

1.0 正式版预告于 2022 年 1 月发布，2 月 2 日发布正式版演示视频，期间 UI 曾经历四次更迭（2024 年 6 月，1.0 精简版发布，大小从 200 多 MB 精简到约 5 MB）。




2022 年 4 月末，工作室确定使用【Tuling 内核】开发 2.0 版本（代号 Hydrogen），通过引入该内核的 VBA 代码增强该作品内部功能的实用性以及减弱其臃肿性（1.0 本体就有 356 张幻灯片…）。

2022 年 6 月，2.0 Alpha 版的出厂引导界面先行预告发布，7 月发布正式版上线预告，预告发布的一段时间后，【Gemini UI】更名为【Taurus UI】，并于 8 月 20 日上线正式版。

Taurus UI 2 引用了来自 Tuling 的诸多丰富实用性功能（如计算器、更改用户名、自定义壁纸、终端、媒体播放、按钮交互等等），在 VBA 的加持下其部分功能实现了从观赏性到实用性的转变，可谓脱胎换骨（与此同时 BUG 也爆发性增长…）。

2023 年 1 月，工作室将【Tuling 4】的部分新功能（诸如锁屏密码、本地文件管理器）引入 Taurus UI, 并于 1 月 29 日发布版本 2.5 Hydrogen。
3 月份，Taurus UI 2 第一个精简版本发布。4 月 29 日发布的 2.6 版本正式将内核从 Tuling 3 升级至 Tuling 4。
自 2023 年至今，通过不断的补丁更新，系统内部越来越多的 BUG 得到修复。一些新功能和彩蛋（例如系统更新）也在后期被添加了进来。
2024 年 10 月末，Taurus UI 2 最后一个精简版本发布。(2.9.4 Lite)
2025 年 1 月 5 日，工作室将停止对 Taurus UI 2 Hydrogen 的技术支持。




Welcome to Avalon.

2023 年 3 月，工作室开始对 PPT OS 新作【Taurus UI Avalon】的制作，并于 3 月 4 日和 6 月 8 日分别发布出厂引导界面先行预告和初次进入桌面预告。该作品属于纯概念系统，计划不使用任何 VBA 代码（使动画可重复播放的代码除外）来模拟真实操作系统的部分功能。该项目目前处于停滞状态。

2024 年，由于计划工程量空前庞大等原因，Lipus Studio 已停止对 Taurus UI Avalon 的开发。




2024 年 7 月，Lipus Studio 基于中途废弃开发的 Tuling 5 Despina 开始制作 Taurus UI Camelot (3.x)，并于 8 月初发布了第一个预览版。Camelot 既补完了源系统未完成的部分 (可实现的部分)，又在源系统 UI 和功能基础上加上了自身特色 (诸如全新毛玻璃应用 UI、可更改设备名功能)，传承创新二者并进。

同时，Camelot 也将是 Taurus UI 最后一个大版本，在该版本的生命周期结束后（2025 年 8 月 14 日停止技术支持），Lipus Studio PowerPoint UI Project 计划也将迎来自己的终点。




更新日志

该更新日志从 2.0 Alpha 开始，由宝船飞瑞秋重新编辑。

  • 3.x

3.1.0 Camelot Official (内部版本 30031)
（由于此次部分文件有所变动，此次更新请从官网处下载，并替换原有文件包 (Taurus UI 2 Hydrogen 用户除外)）
1、修复了 Tuling 开发者工具无法绑定延迟切换宏的问题。
2、给照片一样添加了部分动画。
3、修复了应用商店里的应用无法打开的问题。
4、修复了计算器换算错误的 BUG。
5、修复了网络壁纸第二个 API 源无法获取的问题。
6、修复了恢复模式下载备份报错的问题。
7、修复了更新脚本 Bat 备份出错的问题。
8、细微的修复与改进。

3.0.0 Camelot Official (内部版本 30030)
1、修复了实验性功能无背景主题应用后变成紫色背景的 BUG。
2、添加了将 GIF 设置为壁纸的功能。
3、修复了应用商店里的应用无法打开的问题。
4、修复了相机，计算器，应用商店打开应用的动画 BUG。
5、修复了部分地方点击没有动画的问题。
6、删除部分重复代码。
7、修复了文件应用浏览器的位置。
8、添加了用户手册。
9、修复了浏览器内 WebBrowser 插件未正常显示的问题。
10、细微的修复和改进。

3.0.0 Camelot Gamma Preview (内部版本 30020)
1、更新 Tuling 5 内核版本到 DEMO2 50045。
2、对部分控件输入框和选择框添加了主题色。
3、修复了恢复模式部分地方字体不对的问题。
4、禁用了 OOBE 里的 Tuling 开发者工具 的大多数指令，因为会损坏 OOBE 且大多数指令都会报错。
5、修复了 OOBE 里的部分超链接的问题。
6、现已支持通过网络 API 将锁屏签名改成随机古诗词。
7、给锁屏界面添加了刷新网络壁纸的按钮。
8、给获取网络壁纸添加了失败提示，因为有的时候可能会因为有些原因导致无法获取壁纸例如杀毒软件限制。
9、当 Office 版本号不符合要求版本时，会自动禁用相机应用。
10、修复了 设置-实验性功能-无背景主题 的代码错误问题。
11、修复了部分系统音效缺失的问题（比如关机音效还有错误音效等）。
12、修复了屏幕截图动画播放有误的问题。
13、修复了鸣谢名单部分文字超出形状的问题。
14、修复了计算器点击清除按钮结果执行 cos 的 bug。
15、修复了应用商店部分应用超链接的问题。
16、对系统进入动画进行了一些小修改。
17、为桌面、锁屏分别添加了 3 张壁纸。
18、细微的修复与改进。

3.0.0 Camelot Beta Preview 2 (内部版本 30011)
1、更换了部分 UI (锁屏密码、更多壁纸页…)。
2、对部分 UI 做出调整以更好适配深浅色模式 ( 如修改部分 UI 元素的颜色)。
3、在 设置-关于 处增加评论反馈渠道。
4、修复了新系统彩蛋自动切换幻灯片的 BUG。
5、对禁用网络壁纸的代码进行了修改。
6、修复了切换深色主题后开机界面部分地方颜色被更改的 BUG。
7、修复了部分地方刷新网络壁纸的按钮不显示的 BUG。
8、修复了 设置-通用 里的屏幕分辨率固定为 1920 x 1080 的问题，并添加了重新加载分辨率的按钮。
9、修复了开始菜单上部分快捷入口无法进入的问题。
10、修复了恢复模式终端版本号错误的问题。
11、对使用向导 (OOBE) 的部分 UI 进行细微修改。
12、细微的修复和改进。

3.0.0 Camelot Beta Preview 1 (内部版本 30010)
1、修复了记事本应用的某些问题。
2、对浏览器进行部分改动。
3、修复了部分地方翻译不全的问题。
4、Taurus UI 2 时期的 “快应用” 现在同应用商店合并，商店也已完全开放。
5、给计算器添加了动画。
6、添加了崩溃屏幕。
7、打开控制中心或开始菜单时，用户只需点击屏幕空白处即可返回桌面。
8、在 设置-系统更新 里添加了从官网获取更新的选项，以防下载失败。
9、删除、修改了部分代码。
10、修复了无法设置桌面和锁屏文字颜色的问题。
11、对系统彩蛋进行了更改。
12、实验性功能现已开放。
13、对终端的部分指令进行改动。
14、更新了修复模式的 UI 和部分功能。
15、添加了来自 Tuling 5 Despina 的原生壁纸。
16、修改了部分翻译内容。
17、细微的修复与改进。
（注：该版本不代表正式版，功能在运行过程中仍会存在问题）

3.0.0 Camelot Alpha Preview 2 (内部版本 30002)
1、修复了大量 APP 界面在五彩主题色下颜色错位的问题。
2、修复了部分 VBA 控件不跟随浅深色模式切换变化颜色的问题。
3、更改了部分翻译内容，同时修复了部分翻译内容错位 (A 内容出现在 B 位置) 的问题。
4、对部分 UI 的动画进行优化。
5、对部分内置颜色的 RGB 值进行修改，提升其对比度。
6、修复了系统部分地方超链接无法点击或没有动画的问题。
7、修复了打开计算器应用时任务栏多出来终端图标的 BUG。
8、修复了部分应用 UI 显示异常的问题（如文件应用下 Dock 栏没有显示退出按钮，且无法打开设置应用的问题）。
9、对记事本 VBA 文本框进行语言适配。
10、细节修复和改进。
（注：该版本属于早期预览版，大量功能在运行时会出现效果不佳等情况。大部分问题会在 Beta Preview 版本中得到修复）

3.0.0 Camelot Alpha Preview 1 (内部版本 30001)
1、正式使用 Tuling 5 (内部版本 50041) 作为内核。
2、跟随 Tuling 5 的 UI 样式，对系统和内部程序 UI 进行了大幅度改动，并为内部程序 UI 适配模糊壁纸背景。
3、新增诸多来自 Tuling 5 的特色功能 (如：网络壁纸、更换头像、切换语言、设置主题色…)
4、删除了 Tuling 5 的壁纸系统和 BIOS，继续沿用 Taurus UI 2 的壁纸系统和 BIOS。
5、删除了日历、时钟等等 Taurus UI 2 的部分内部程序。
（注：该版本属于早期预览版，大量功能在运行时会出现效果不佳等情况。大部分问题会在 Beta Preview 版本中得到修复）
